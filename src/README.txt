The project “Hotel Booking” is about the prebooking of the rooms provided by the “IDFC” hotel to the interested customers.
The respective project has been developed using Python, Flask, HTML, JavaScript, CSS and MYSQL.
The python language has been used for backend.
The flask is used as a server.
HTML, JS and CSS for designing the User Interface.
The MYSQL database has been used for database purpose.
The “Hotel Booking” project provides user to do the booking of the rooms online. It is the graphical user interface project.
The user is asked to enter information to confirm the booking like name, date of booking etc.
